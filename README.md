# taha_dursun_case

### Pre-requisites
* Java
* Maven
* [Allure](https://docs.qameta.io/allure/#_installing_a_commandline)
* IntelliJ IDEA


### Steps
1. Clone this project
2. Open the project in Intellij IDEA
3. Run TestNG.XML
4. Open the terminal in Intellij IDEA
5. Execute **`allure serve allure-results`**
